# Support

Use GitHub Discussions when available for integration questions and GitHub Issues for reproducible bugs or feature proposals.

Before requesting support:

1. read the README and architecture documentation;
2. confirm the issue with the latest version;
3. run `npm run check`, `npm test`, and `npm run build`;
4. provide a minimal reproduction and relevant configuration;
5. remove tokens, guild IDs, and other sensitive data.

Security vulnerabilities must follow `SECURITY.md` and must not be reported publicly.
